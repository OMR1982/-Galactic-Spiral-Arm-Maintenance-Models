# Galactic Spiral Arm Maintenance Models

This repository contains the Python scripts developed for the research paper.

## Repository Contents

- **[code_01_torque_budget](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_01_torque_budget)**: Calculates the dissipative counter-torque ($\tau_{ejecta}$) vs. kinematic shear torque ($\tau_{wind}$).
- **[code_02_monte_carlo](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_02_monte_carlo)**: 3D geometric Monte Carlo model to validate the momentum asymmetry fraction ($f_{asym}$).
- **[code_03_sensitivity_sweep](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_03_sensitivity_sweep)**: Sensitivity analysis of $f_{asym}$ relative to the cooling radius ($R_{cool}$).
- **[code_04_m82_m42_application](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_04_m82_m42_application)**: Numerical application of the model to M82 and M42.
