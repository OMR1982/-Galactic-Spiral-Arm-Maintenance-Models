# Galactic Spiral Arm Maintenance Models

Welcome to the companion repository for my research paper. This repository contains the numerical models and Python scripts used for the analysis.

## Repository Structure & Usage

Each directory corresponds to a specific section/analysis in the research paper:

1. **[code_01_torque_budget](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_01_torque_budget)**: 
   Calculates the dissipative counter-torque ($\tau_{ejecta}$) vs. kinematic shear torque ($\tau_{wind}$).

2. **[code_02_monte_carlo](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_02_monte_carlo)**: 
   Monte Carlo simulations for parameter uncertainty validation.

3. **[code_03_sensitivity_sweep](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_03_sensitivity_sweep)**: 
   Sensitivity analysis of the model parameters.

4. **[code_04_m82_m42_application](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_04_m82_m42_application)**: 
   Application of the developed model to M82 and M42 galaxies.
