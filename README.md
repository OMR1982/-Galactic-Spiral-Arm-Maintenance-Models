## Repository Contents
- [code_01_torque_budget](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_01_torque_budget): Calculates the dissipative counter-torque (tau_ejecta) vs. kinematic shear torque (tau_wind).
- [code_02_monte_carlo](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_02_monte_carlo): 3D geometric Monte Carlo model to validate the momentum asymmetry fraction (f_asym = 5.6% ± 8.9%).
- [code_03_sensitivity_sweep](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_03_sensitivity_sweep): Sensitivity analysis of f_asym relative to the Sedov-Taylor cooling radius (R_cool), generating Figure 2.
- [code_04_m82_m42_application](https://github.com/OMR1982/Galactic-Spiral-Arm-Maintenance-Models/tree/main/code_04_m82_m42_application): Numerical application of the analytical model to the starburst galaxy M82 and Orion Nebula M42, generating Figure 3.
